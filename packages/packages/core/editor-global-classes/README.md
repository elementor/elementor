# Editor Global Classes

> [!NOTE]
> This documentation is a placeholder and will be expanded with detailed usage examples and API reference.

## Overview

The `@elementor/editor-global-classes` package provides global CSS classes management and application for atomic widgets and the Elementor editor. It handles global class definition, management, application to atomic widgets, and style inheritance.

## Features

- Global class definition and management
- Class application to atomic widgets
- Style inheritance from global classes
- Integration with atomic widget style system

## Integration with Atomic Widgets

This package is used by atomic widgets to support global CSS classes. See the [Atomic Widgets documentation](../../../../modules/atomic-widgets/README.md) for more information on how global classes are used in atomic widgets.
