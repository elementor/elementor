# Editor Variables

> [!NOTE]

## Overview

The `@elementor/editor-variables` package provides CSS variables management for atomic widgets and the Elementor editor. It handles variable scoping, value resolution, inheritance, and integration with the atomic widget style system.

## Features

- CSS variable management and scoping
- Variable value resolution and inheritance
- Integration with atomic widget style system
- Dynamic variable updates and reactivity

## Integration with Atomic Widgets

This package is used by atomic widgets to manage CSS variables for dynamic styling. See the [Atomic Widgets documentation](../../../../modules/atomic-widgets/README.md) for more information on how variables are used in atomic widgets.
